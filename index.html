Pasang Aplikasi Manjur di HP

Bisa diinstal di HP Android & berjalan offline

Instal Cepat 1-Klik (PWA Android)

Aplikasi ini telah dilengkapi konfigurasi manifest.json & Service Worker dengan ikon resmi Manjur Laundry.

*Jika tombol tidak memicu dialog otomatis, ikuti panduan Chrome di bawah ini.

Cara Pasang di HP Android (Google Chrome)

  1. 1

    Buka aplikasi ini di browser Google Chrome pada HP Android Anda.

  2. 2

    Ketuk ikon Titik Tiga (⋮) di pojok kanan atas browser Chrome.

  3. 3

    Pilih menu "Tambahkan ke Layar Utama" (Add to Home Screen) atau "Instal Aplikasi".

  4. ✓

    Selesai! Ikon Manjur Laundry muncul di menu HP Anda & bisa dibuka full-screen seperti aplikasi asli.

Distribusi File APK Mandiri (.apk)

Aplikasi ini sudah siap dibungkus menjadi file .apk Android. Anda dapat menggunakan layanan gratis seperti PWABuilder.com atau Website 2 APK Builder dengan memasukkan URL aplikasi ini beserta file manifest.json & logo logo_manjur.png.
